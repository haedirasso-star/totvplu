# totvplu — موقع تحميل تطبيق توتيفي TOTV+

موقع التحميل الرسمي لتطبيق توتيفي، يعمل على GitHub Pages مع الدومين `totv.app`.

## الملفات
- `index.html` — الصفحة الرئيسية (أفلام، نجوم، كرة قدم، أسعار، تواصل).
- `download.html` — صفحة التحميل: زر **أندرويد** وزر **شاشات سمارت** بتصميم حصري وخلفية بوسترات.
- `logo.png` / `logo44.png` / `Icon-*.png` — الشعار والأيقونات.
- `CNAME` — الدومين `totv.app`.
- `sitemap.xml` / `app-ads.txt` — ملفات SEO.

## روابط التحميل
تستخدم الصفحة رابط **latest** الذي يشير دائماً لأحدث إصدار:
`https://github.com/haedirasso-star/totvplu/releases/latest/download/<الملف>`

هذا يحل مشكلة الروابط المعطّلة نهائياً (لا يتأثر بحالة الأحرف في اسم الوسم مثل v1 / V1).

## كيف ترفع نسخة شاشات سمارت منفصلة لاحقاً
1. أنشئ إصداراً جديداً (Release) وارفع فيه ملف باسم `totvplus-tv.apk`.
2. في `download.html` غيّر السطر:
   `const TV_FILE = 'totvplus.apk';`  ←  إلى  `const TV_FILE = 'totvplus-tv.apk';`
3. احفظ، وسيعمل زر شاشات السمارت تلقائياً على الملف الجديد.

> ملاحظة: حالياً كِلا الزرّين يحمّلان `totvplus.apk` العامل (نسخة أندرويد تعمل على Android TV)، فلا توجد روابط معطّلة.
